# exvis

exvis **vis**ualizes MIP instances based on their **ex**pressions (variables and constraints) as a graph.

## Installation

```bash
pip install exvis
```

## Usage

```bash
exvis --input data/example/enlight_hard.mps.gz
```

For further options, see `exvis --help`.
